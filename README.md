# 观迹 | WatchLog

一个自部署、轻量、可扩展的个人观影记录项目。

## 当前状态

- 使用 Nuxt 4 作为主应用框架
- 计划提供 SPA 主界面与可嵌入的 Web Component Widget
- 数据源首期接入豆瓣，后续保留 TMDB 扩展能力
- 存储方案使用 SQLite

## 开发命令

```bash
pnpm install
pnpm dev
pnpm build
pnpm preview
```
